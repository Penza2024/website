Uses

Hardware, software, and tools I reach for every day.

Hardware

  • MacBook Pro
    Primary machine for everything — iOS dev via Xcode, React Native builds, and running Docker stacks locally.
    daily driver
  • ESP32-WROOM-32 (38-pin)
    The microcontroller of choice for IoT projects. Used in the Smart Grid System and Predictive Maintenance Node. Runs C++ firmware over the Arduino framework.
    hardware
  • EBYTE E220 LoRa Module
    868/915 MHz LoRa transceiver used for long-range, low-power sensor communication. The reason the E220 Configurator GUI exists.
    hardware
  • DHT11 / SW-420 / KY-038
    Temperature & humidity, vibration, and sound sensors wired to the ESP32 Predictive Maintenance Node. All run on 3.3V.
    sensors

Editor & Terminal

  • VS Code
    Primary editor for TypeScript, React Native, Python, and C++. Extensions: ESLint, Prettier, Tailwind CSS IntelliSense, Arduino.
    editor
  • Xcode
    Required for SwiftUI development and signing iOS builds. Also the simulator for React Native iOS testing.
    editor
  • zsh + oh-my-zsh
    Shell of choice. Plugins: git, z (jump directories), syntax-highlighting.
    terminal

Software & Apps

  • Docker Desktop
    Running ChirpStack, MySQL, and multi-service stacks locally. Also the local Kubernetes cluster for Incredibulls testing.
    devops
  • Postman
    API testing and collection management across Supabase edge functions, Node.js APIs, and ChirpStack webhooks.
    dev tool
  • MetaTrader 5
    Platform for running and backtesting the NNFX Trading Robot. MQL5 is its scripting language.
    trading
  • Expo Go
    For live-reloading React Native apps on a physical device during development of Reservit and BusTicKTeT.
    mobile
  • TablePlus
    GUI for browsing Supabase/PostgreSQL and MySQL databases. Cleaner than psql for exploratory queries.
    database

Stack defaults

  • TypeScript
    Default language for any new web or mobile project. The type safety catches enough real bugs to be worth the overhead every time.
    language
  • Supabase
    Backend of choice for mobile apps — PostgreSQL, auth, edge functions, and realtime out of the box. Replaced Firebase for most new projects.
    backend
  • Expo (managed workflow)
    React Native without the native configuration pain. EAS Build handles CI/CD for mobile.
    mobile
  • Vite
    Default bundler for React web projects. The dev server speed is hard to go back from.
    web

Uses

Hardware, software, and tools I actually use — sourced from my Applications folder.

Machine

  • macOS (OpenCore patched)
    Running macOS Sequoia on patched hardware via OpenCore Legacy Patcher. Macs Fan Control keeps temps in check.
    daily driver
  • Raspberry Pi
    Used for network experiments and as a local server. Raspberry Pi Imager handles OS flashing.
    hardware
  • ESP32-WROOM-32 (38-pin)
    Microcontroller of choice for IoT projects — Smart Grid System and Predictive Maintenance Node. Firmware flashed via CP210x USB serial driver.
    hardware
  • EBYTE E220 LoRa Module
    868/915 MHz LoRa transceiver for long-range, low-power sensor communication. The direct reason the E220 Configurator GUI exists.
    hardware
  • DHT11 · SW-420 · KY-038
    Temperature/humidity, vibration, and sound sensors used in the Predictive Maintenance Node. All run on 3.3V GPIO.
    sensors

Editors

  • VS Code
    Primary editor. TypeScript, React Native, Python, C++. Claude Code extension runs inside it for AI-assisted work.
    editor
  • Cursor
    AI-first fork of VS Code. Used when I want inline generation and multi-file edits with full context awareness.
    editor
  • Windsurf
    Codeium's agentic IDE. Another AI editor I rotate through depending on the task.
    editor
  • Xcode
    Required for SwiftUI development, iOS app signing, and the React Native iOS simulator.
    editor
  • Arduino IDE
    For writing and flashing C++ firmware to ESP32 boards. Used heavily in IoT projects alongside PlatformIO.
    firmware
  • zsh
    Shell of choice on macOS. Standard setup, nothing fancy.
    terminal

AI Tools

  • Claude
    Daily driver for reasoning, writing, and code review. The desktop app and Claude Code CLI are both open most of the day.
    ai
  • Ollama + GPT4All
    Local LLM inference. Used when I need to run models offline or experiment with smaller open-weights models without sending data to an API.
    ai · local
  • Kiro
    AWS's AI IDE — exploring spec-driven development where the model plans before it codes.
    ai · ide
